## POMIDOROWY SEN

Pierwszy projekt z użyciem HTML, CSS(RWD), PHP, MYSQL:
System newsów i produktów z mini panelem administracyjnym pozwalającym na:
dodawanie, usuwanie, edycję i wyświetlanie newsów, produktów z obrazkami.
